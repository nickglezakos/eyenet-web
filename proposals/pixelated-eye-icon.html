Pixelated Eye Icon — Design Proposal

A rect-based, pixelated eye icon for Eyenet, derived from the wordmark's design DNA.

1. Design DNA — Analysis of Existing Wordmark

Reference: Current Wordmark (Light Mode)

Light background

Construction Rules Extracted

  • Primitives: Axis-aligned <rect> elements only — no curves, no paths, no circles.
  • Grid unit: 8 px base width; heights vary (8, 13, 14, 25, 35, 52, 60) for organic shaping.
  • Colour cycling: Three blues rotate across elements, creating visual rhythm and depth.
  • Opacity modulation: Values range from 0.7 to 1.0, giving each cell a subtly different weight — the hallmark "pixel shimmer" effect.
  • No strokes or gaps: Rects touch edge-to-edge; negative space defines the shape.

2. Proposed Concepts

Three icon concepts at different fidelity levels, all built from the same design DNA. Each uses only <rect> elements, the 3-blue palette, and opacity modulation.

A balanced almond-shaped eye on a 14 × 10 cell grid (112 × 80 px at 8 px/cell). Wide enough to read as an eye at any size, detailed enough for hero placement. Proportions: 1.4 : 1.

5. Implementation Notes

File Naming Convention

Following the existing structure in assets/logos/:

  • svg/eye-icon-light.svg — Concept A, light-mode palette
  • svg/eye-icon-dark.svg — Concept A, dark-mode palette
  • svg/eye-icon-compact-light.svg — Concept B, light-mode
  • svg/eye-icon-compact-dark.svg — Concept B, dark-mode
  • png/eye-icon/light/ — PNG exports at 32, 64, 128, 192, 512 px
  • png/eye-icon/dark/ — PNG exports at same sizes
  • svg/eye-icon-hires-light.svg — Concept D, light-mode
  • svg/eye-icon-hires-dark.svg — Concept D, dark-mode
  • svg/eye-icon-chunky-light.svg — Concept E, light-mode
  • svg/eye-icon-chunky-dark.svg — Concept E, dark-mode
  • favicon/ — ICO and PNG from Concept B or E

Animation Potential

The rect-based construction enables simple CSS/SMIL animations without path complexity:

  • Blink: Collapse upper and lower lid rows toward centre (translate Y + height to 0), then restore.
  • Pulse: Cycle opacity values on iris rects in sequence (wave sweep left-to-right).
  • Focus: Scale pupil from 2×2 to 3×3 cells (expand rects outward from centre).
  • Scan: Sweep a bright highlight rect across the iris ring.

Colour Theming

Every rect uses a fill from the shared palette. To re-theme for a different brand colour, replace the three base hues globally — the opacity variation and grid structure carry the visual effect regardless of hue.
